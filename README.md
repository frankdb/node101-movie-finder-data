# Node 101 Movie Finder Data

This project creates a cache server to store movie data from the OMDB API.

## Getting Started

Clone repo into local directory. Run "npm start" and open on localhost:3000.

### Installing

Run "npm install" in Terminal

## Running the tests

Run "npm test"

## Built With

- Node

- Express
